Welcome to my digital identity. This repository houses the source code for my personal portfolio, designed under the concept of **"Reforging Phase"**. It represents a shift from random experimentation to intentional, system-oriented engineering.

## 🛠️ Tech Arsenal (No Frameworks)
This system is built on pure fundamentals to ensure maximum control and performance.

| Module | Technology | Usage |
| :--- | :--- | :--- |
| **Skeleton** | HTML5 (Semantic) | Structure & Accessibility |
| **Skin** | CSS3 (Native Variables) | Glassmorphism, Grid, Animations |
| **Logic** | Vanilla JavaScript (ES6+) | Scroll Observer, Smooth State |
| **Assets** | SVG & Canva | Visual Fragments |

## 💎 Key Features
* **Aurora Engine:** Custom CSS-based ambient background animation.
* **Holographic UI:** Glassmorphism design system with dynamic lighting.
* **Scroll Reveal:** Intersection Observer API for cinematic element entry.
* **Navigation Dock:** Floating macOS-style dock with active radar tracking.
* **Chronological Log:** Zig-zag timeline layout for history tracking.

## 🚀 How to Initialize (Run Locally)

If you wish to inspect the system architecture locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Stylenecy/dex-portfolio.git](https://github.com/Stylenecy/dex-portfolio.git)
    ```
2.  **Navigate to directory**
    ```bash
    cd dex-portfolio
    ```
3.  **Launch**
    Open `index.html` in your browser (or use Live Server in VS Code).
